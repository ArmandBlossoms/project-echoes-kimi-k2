# Domain Docs

How the engineering skills should consume this repo's domain documentation when exploring the codebase.

## Before exploring, read these

- **`PLAN.md`** at the repo root — the full plan. Read this first before anything substantive.
- **`CLAUDE.md`** at the repo root — conventions, stack, critical rules, VFCC design system tokens, tone rules, build phases.
- **`.review/CONSOLIDATED-FINDINGS-v2.md`** — council audit findings from the Claude Opus 4.7 + Codex planning session. Read instead of re-running the adversarial council.
- **`docs/superpowers/plans/`** — phase-by-phase implementation plans (generated by the `writing-plans` skill). Read the plan for the phase you are about to build.

There is no separate `CONTEXT.md`. `PLAN.md` and `CLAUDE.md` jointly serve that role for this project.

## File structure

Single-context repo:

```
/
├── PLAN.md                      — canonical build plan (source of truth)
├── CLAUDE.md                    — agent-facing conventions, stack, critical rules
├── PROGRESS.md                  — live phase tracker
├── README.md                    — public-facing project summary
├── .review/
│   ├── CONSOLIDATED-FINDINGS-v2.md
│   └── ...                      — council audit artefacts
├── docs/superpowers/plans/      — per-phase implementation plans
├── docs/agents/                 — mattpocock/skills configuration
├── src/                         — application code
└── ...
```

## Use the glossary's vocabulary

When your output names a domain concept (in an issue title, a refactor proposal, a test name), use the term as defined in `PLAN.md` and `CLAUDE.md`. Do not drift to synonyms those files explicitly avoid.

Key terms to get right:
- **"care-experienced children and young people"** — never "looked-after", "LAC", "service users", "care leavers"
- **"VFCC"** — Voices From Care Cymru. Never "VFCCymru" or "VFC".
- **"voices"** — individual contributions to the dashboard. Never "users", "records", "tickets".
- **"Echoes · Atseiniau"** — the product name, in Shrikhand, with Welsh circumflexes.
- **"per cent"** in prose; `%` only in chart labels.

## Flag conflicts

If your output contradicts an existing decision in `PLAN.md` or `CLAUDE.md`, surface it explicitly rather than silently overriding. This project has 23 critical rules that are non-negotiable.
